核心功能

  • 生物识别与密码双重验证
  • 智能密码生成器
  • 密码安全强度检测
  • 自动备份与恢复
  • 快速搜索与分类管理

字符大师

像素之美，字符之趣

一款创意图像处理微信小程序，能将普通图片转换为由ASCII字符组成的艺术作品。支持多种参数调整，让您创作出独一无二的字符艺术。

核心功能

  • 图片到ASCII字符转换
  • 自定义字符集与转换参数
  • 彩色/黑白模式切换
  • 结果保存与分享
  • 实时预览效果
技术栈： 微信小程序 Canvas JavaScript 图像处理算法

ASCII艺术桌面版

字符构建的视觉艺术

基于Qt开发的桌面ASCII艺术转换工具，支持图片和视频的高级处理。通过集成FFmpeg视频框架，实现了视频到ASCII艺术的实时转换。

核心功能

  • 视频到ASCII实时转换
  • 高精度图片处理
  • 多参数自定义选项
  • 跨平台桌面支持
  • 视频帧提取与处理
技术栈： Qt C++ FFmpeg 多媒体处理 并发计算

UDP交流互动软件

创意共享，即时连接

基于Qt开发的网络交流平台，采用UDP协议实现高效通信，集成绘图、照片处理、即时对话和用户系统于一体，为用户提供全方位的交互体验。

核心功能

  • 实时共享绘图画布
  • 图像编辑与处理
  • 低延迟即时通讯
  • 用户账号管理
  • UDP高效数据传输
技术栈： Qt C++ UDP协议 网络通信 多线程

微积分计算器

复杂数学，简单解析

基于MATLAB开发的高级微积分计算工具，支持复杂函数的求导、积分、极限计算及图形可视化，为数学学习和科研提供强大的工具支持。

核心功能

  • 多元函数导数计算
  • 定积分与不定积分求解
  • 函数图形可视化
  • 微分方程数值解法
  • 计算步骤详细解析
技术栈： MATLAB 数值计算 符号计算 数据可视化 GUI设计
